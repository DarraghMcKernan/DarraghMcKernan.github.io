Drawing Group.

My concept feature is a group chat room that will

have a chat that all participants can use and a

'whiteboard' that all users that the admin has

allowed to can draw on. Each user can only delete

their own drawings but the admin can delete all

drawings. This would hopefully prevent people

removing work on purpose or by mistake. It might

be useful for teams trying to brainstorm or draw

concepts that everyone can see in real time.
